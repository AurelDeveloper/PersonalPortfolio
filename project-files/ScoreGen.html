Description

The program is designed to calculate the grades of a class, and one should be able to determine the average of all grades.

Code Description

The code first calculates the average, from which the standard deviation can be calculated. With this, outliers are filtered and stored in a separate array. Using the standard deviation, the point at which the grade calculation threshold changes is calculated. For example, 0 points should still result in a grade of 1. After that, the grades are calculated (this part was programmed by my father).
