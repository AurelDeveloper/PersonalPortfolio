Beschreibung

Das Programm sollte die noten eine Klasse ausrechnen und man sollte den Durchschnitt aller Note bestimmen können.

Code Beschreibung

Der code rechnet zuerst den Durchschnitt aus diesem kann man die Standart deviation ausrechnen mit diesem werden die Ausreisser gefiltert und in einem Separaten Array gespeichert, mit der Standart deiviation kann man auch noch der Punkt ausrechnen bei dem die Noten berechnungs schwäle sich ändert so das die Schlächten Aussreisser immer noch ihre richtige Noten griegen Beispiel: 0 Punkte sollte immer noch eine 1 ergeben, danach werden die Noten ausgerechnet (dieser Teil hat mein Vater programmiert)

Herausforderungen

Lösungen

Die Mathematischen Konzepte hat mir mein Vater erklärt. Das einarbeiten habe ich teilwese mit Kommentaren und saubereren Code gelöst.
